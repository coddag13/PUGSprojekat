\# PUGSprojekat



Web aplikacija za planiranje putovanja.



\## Struktura projekta



\- `frontend/travel-planner` - React frontend aplikacija

\- `backend/TravelPlanner` - backend dio aplikacije

\- `docs` - dokumentacija, arhitektura i dijagrami
